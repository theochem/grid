About
-----
GRID is the legacy HORTON 2 numerical integration module.

License
-------

GRID is distributed under GPL License version 3 (GPLv3).


Installation
------------

To install GRID, use conda:

```
conda install -c theochem grid
```


Testing
-------

To run tests:

```
nosetests -v grid
```
