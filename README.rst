About
-----
Old_Grids is the legacy HORTON 2 grid module.

License
-------

Old_Grids is distributed under GPL License version 3 (GPLv3).


Installation
------------

To install Old_Grids, use conda:

```bash
conda install -c theochem old_grids
```


Testing
-------

To run tests:

```bash
nosetests -v old_grids
```
